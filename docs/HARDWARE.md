# 硬件指南：盘、硬盘盒、接口

## 一句话配置

**1TB NVMe SSD（TLC、带缓存更佳）+ USB4/雷电硬盘盒（带散热片）**，是体验与价格的平衡点。预算紧就 USB 3.2 Gen2 (10Gbps) 硬盘盒——正好打平上游作者的已验证基线。

## 接口速度对照（决定冷态等待）

引擎冷态每 token 要从盘里读 ~11GB（75 层 × 8 专家），速度几乎线性取决于随机读带宽：

| 链路 | 实际随机读 | 冷态 decode | 判定 |
|---|---|---|---|
| USB 3.0 (5Gbps) | ~0.4 GB/s | ~0.02–0.05 tok/s | 不可用级别，别买 |
| **USB 3.2 Gen2 (10Gbps)** | ~0.9–1 GB/s | **~0.05–0.1 tok/s** | **作者开发机基线（WSL2+VHDX 实测）** |
| USB 3.2 Gen2x2 (20Gbps) | ~1.6–2 GB/s | 好于基线 | 需要主板也支持 Gen2x2，否则掉到 10Gbps |
| **USB4 / 雷电 3/4** | **~2.5–3.5 GB/s** | 明显更好 | **推荐** |
| 原生 PCIe4 ×4 | 3.5–5 GB/s | ~0.5 tok/s 量级 | 台式机内置对照组 |
| 原生 PCIe5 ×4 | 8–10+ GB/s | 社区实测 O_DIRECT 8.81–10.51 GB/s → 0.28–0.41 tok/s（此时瓶颈转回 CPU matmul） | 内置对照组 |

注意瓶颈转移：盘越快，CPU matmul 越快成为新瓶颈（上游 AVX2 内核 ~250 GFLOP/s，每 token ~80 GFLOP）。外置方案里盘基本永远是瓶颈，所以**预算优先给接口和硬盘盒**。

## 硬盘盒选购要点

1. **主控**：USB4/雷电选 ASM2464PD；10Gbps 选 RTL9210B / ASM2364。避开无型号杂牌。
2. **散热**：持续满载读取会让廉价盒子/QLC 盘过热降速。选带鳍片/导热垫的金属盒。上游 README 也明确提醒 sustained thermals 是真实风险。
3. **线材**：USB4 用认证 40Gbps 线；差线会掉到 10Gbps。
4. **盘**：1TB TLC NVMe 即可（模型 370GB + 系统余量）。QLC 也能用（读取场景），但持续读温控略差。**读取基本不磨损 SSD**——上游确认流式加载是纯读；真正伤盘的是系统 swap（所以别让机器内存见底）。

## 到手后必做：实测

```bash
scripts/iobench_check.sh --ssd /mnt/myssd
```

它用上游 iobench 复现引擎的真实 I/O 模式（19MB × 64 并行随机读，含 O_DIRECT），并按上面的基线给出四档判定。商家标称速度不算数，这个 5 秒的测试才算数。
